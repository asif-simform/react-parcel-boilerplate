# React Parcel

React + Parcel boilerplate

## Features:
* Parcel 1.10.0
* Babel 7
* React 16.12.0
* TypeScript 3.7.0
* MobX 5.15.1
* styled-components 4.4.1
* React Router 5.1.2
* Sass
* ESLint
* Husky + Lint-staged

* **dev**: Run dev build with server instance,
